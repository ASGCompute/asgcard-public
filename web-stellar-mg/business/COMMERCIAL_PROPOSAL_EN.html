What is ASG Card

ASG Card is payment-to-card infrastructure for AI agents. It converts paid API actions into immediate virtual card issuance and funding so agents can execute spend safely and programmatically.

x402-native Virtual Cards API USDC on Stellar x402 facilitator path

How it works

  1. 1. Agent requests a paid action: call card create/fund endpoint.
  2. 2. x402 challenge + payment: x402 path verifies payment intent and execution.
  3. 3. Settlement: USDC settles on Stellar and payment is confirmed.
  4. 4. Card response: ASG Card issues/funds card and returns details plus controls.
